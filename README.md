## 설명
이 프로젝트는 C#과 WPF를 사용하여 구현된 SnakeGame 입니다. 사용자는 뱀을 조작하여 화면에 나타나는 먹이를 먹으며 뱀을 성장시킬 수 있습니다. 뱀이 벽이나 자신의 몸에 부딪히면 게임이 종료됩니다.

## 기능
- 화살표 키를 사용한 뱀의 이동
- 먹이를 먹을 때마다 뱀의 길이 증가
- 벽이나 몸과 충돌 시 게임 종료
- 점수 표시 기능
- 게임 리셋 기능
